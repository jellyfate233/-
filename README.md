# 程序用途
用于将一段视频进行边缘提取,并实时输出为视频的demo
# 运行环境
* Linux (tested on Ubuntu 20.04.1 LTS)
* Python 3.7 (tested on 3.6.15)
* opencv 4.5.3
# 使用说明
* 视频处理前将视频路径输入到程序<视频边缘检测实时处理v1.py>的视频载入path中。
* 运行程序，输出转换视频，结果保留到当前路径
